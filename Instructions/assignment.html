Blender

Blender is an open-source 3D Modelling and Animation (and much more) software. Many topics covered in the lectures are put into practice in this type of software.

In this individual assignment, you are asked to create and render your own 3D model using Blender.

We have included a guide to help you get acquainted with Blender and show how you can use it to model simple 3D shapes.

There are many online Blender tutorials, for example, from the Blender webpage https://www.blender.org/support/tutorials/

We also highly recommend the Donut guide by BlenderGuru: https://www.youtube.com/watch?v=TPrnSACiTJ4

Make sure you follow Blender 2.8 tutorials, there are significant differences from 2.7 or below.

You may also use other modelling software, but make sure you are able to export to OBJ format and render an image of your model.

Rules

You can model a character, a house, a plant, a spaceship, a car or anything else you wish. There are no strict restrictions on the model that you may create as long as it is not too simple, such as a few cubes or spheres. Try to include at least some texture or material to your model.

Following a tutorial (such as the Donut) and making small modifications is also valid for completing this assignment.

Submission

  • One image of your model rendered in Blender - name you file “assignment-blender.png”
  • A zip file including all modelling files (obj, mtl, textures, blender …)
  • A text file (.txt) listing any external references and sources used. In case you modelled everything yourself, just state so in this file.

Important:

  • Do not bundle everything into one file. Upload the image, the zip file, and the text file (if any) separately
  • Make sure you name your image as specified above, otherwise the script will miss your file and it will not be graded
